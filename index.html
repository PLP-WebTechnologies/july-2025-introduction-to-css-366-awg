Box Model Components

Every element is a rectangular box with:

  1. Content (text/images)
  2. Padding (inner space)
  3. Border (frame around element)
  4. Margin (outer space)

Margin Collapsing

Vertical margins between adjacent elements collapse into the larger margin value.

Box 1 (margin-bottom: 40px)
Box 2 (margin-top: 30px)

Box-Sizing Property

Default: box-sizing: content-box; (border/padding add to width)

Better practice: box-sizing: border-box; (border/padding included in width)
